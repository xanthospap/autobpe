Chapter 1. The AutoBpe Software Package

Purpose

The purpose of the AutoBpe Software Package is to assist the routine prossesing of GNSS data at DSO. With time, it has grown to support a wide range of activities connected (either directly or not) with the analysis of such data. The package programs support (but are not limited to):

  • downloading GNSS data and products,

  • preproccesing rinex data (via GNSS),

  • proccessing GNSS data via Bernese software,

  • compiling summary files, reports, graphs, time-series files, etc,
