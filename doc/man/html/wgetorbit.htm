NAME

wgetorbit − AutoBpe Software Collection

INTRODUCTION

AutoBpe Software Collection is a collection of tools that accomodates automatic processing of GNSS data mainly via Bernese GNSS.

SYNOPSIS

wgetorbit [OPTIONS]... −y [YEAR] −d [DAY OF YEAR]

DESCRIPTION

Download GNSS orbit files, in sp3 format. GNSS considered are GPS and GLONASS. Note that some Analysis Centers have multi-GNSS orbit files (e.g. CODE) while others maintain different files for each GNSS.

Mandatory arguments for long options are mandatory for short options too.
−a, −−analysis-center

specify analysis center; valid analysis centers are igs and cod

−t, −−type

specify the product type; this can be: f for final, r for rapid and u for ultra-rapid. Note that if -t is not turned on (default), then the script will search for whatever type is available (starting from final,then rapid and then for ultra-rapid) and download the first file that it finds. Note also, that GLONASS orbit files are not available as rapid. In case two files must be downloaded (for gps and glonass), then the two files will be merged to one mixed orbit file, named as the original gps sp3. E.g. running wgetorbit.sh -a igs -t f [...] the resulting file will be igsWWWWD.sp3.Z which will contain the respective iglWWWWD.sp3 file. For a detailed description of the distinct cases, see the EXPLICIT CASES table.

−y, −−year

specify year (4-digits)

−d, −−doy

specify day of year

−o, −−output-directory

specify output directory where the downloaded files are going to be saved (default ./)

−r, −−force-remove

if a file is found with the same name as the one to be downloaded, it will be deleted before downloading. The default behaviour is that if a file is found with the same name as the one to be downloaded, then the file will NOT be downloaded but counted as downloaded.

−z, −−decompress

decompress the downloaded files (default do not decompress).

−n, −−upper-case

truncate all downloaded files to upper case letters.

−h, −−help

only print the help message and exit

−s, −−standard-names

if this switch is turned on, then the downloaded file will be renamed to the standard naming convention (see the EXPLICIT CASES table)

EXIT STATUS

The program will return an integer, denoting the exit status. There are two distinct cases,

•

Exit Status = -1; this denotes that an error has occured

•

Exit Status >=0 ; successeful run

DEPENDANCIES

getopt, bpepy.gpstime, bpepy.products, python, read The python library bpepy.gpstime must be installed and set in the python path variable. the same holds for the python bpepy.products library.
